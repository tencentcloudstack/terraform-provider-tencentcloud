# tencentcloud_teo_export_zone_config

Provides a resource to export TEO (TencentCloud Edge One) zone configuration.

This resource allows users to export zone configurations for backup, audit, or migration scenarios.

## Example Usage

```hcl
resource "tencentcloud_teo_zone" "example" {
  zone_name = "terraform-test.example.com"
  type      = "generic"
}

resource "tencentcloud_teo_export_zone_config" "example" {
  zone_id = tencentcloud_teo_zone.example.id
}
```

## Argument Reference

The following arguments are supported:

* `zone_id` - (Required, ForceNew, String) Zone ID.
* `types` - (Optional, Set: [`String`]) List of types to export. If not specified, all types will be exported. Current supported values include:
  * `L7AccelerationConfig`: Export L7 acceleration configuration, corresponding to "Site Acceleration - Global Acceleration Configuration" and "Site Acceleration - Rule Engine" in the console.

## Attributes Reference

In addition to all arguments above, the following attributes are exported:

* `content` - (String) The specific content of the exported configuration. Returned in JSON format, encoded in UTF-8.
* `request_id` - (String) The unique request ID, generated by the server.
* `id` - (String) Zone ID.

## Import

This resource can be imported by using the `zone_id`. For example:

```shell
terraform import tencentcloud_teo_export_zone_config.example zone-id-xxx
```
