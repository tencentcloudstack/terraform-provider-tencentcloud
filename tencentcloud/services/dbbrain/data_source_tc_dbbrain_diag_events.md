Use this data source to query detailed information of DBbrain diag events

Example Usage

```hcl
data "tencentcloud_dbbrain_diag_events" "example" {
  start_time = "2025-01-01T10:41:59+08:00"
  end_time   = "2026-12-31T00:00:00+00:00"
}
```

Or

```hcl
data "tencentcloud_dbbrain_diag_events" "example" {
  start_time = "2025-01-01T10:41:59+08:00"
  end_time   = "2026-12-31T00:00:00+00:00"
}
```
