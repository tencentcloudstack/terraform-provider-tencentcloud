Provides a resource to create a apm apm_prometheus_rule

Example Usage

```hcl
resource "tencentcloud_apm_prometheus_rule" "apm_prometheus_rule" {
}
```

Import

apm apm_prometheus_rule can be imported using the id, e.g.

```
terraform import tencentcloud_apm_prometheus_rule.apm_prometheus_rule apm_prometheus_rule_id
```
