Provides a resource to create a bh bh_auth_mode_setting_config

Example Usage

```hcl
resource "tencentcloud_bh_auth_mode_setting_config" "bh_auth_mode_setting_config" {
}
```

Import

bh bh_auth_mode_setting_config can be imported using the id, e.g.

```
terraform import tencentcloud_bh_auth_mode_setting_config.bh_auth_mode_setting_config bh_auth_mode_setting_config_id
```
